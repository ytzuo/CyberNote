# CyberNote

CyberNote 是一个基于 WPF (.NET 8) 开发的现代轻量级桌面便签应用。它支持多种便签类型，提供活动热力图统计，并支持多种颜色主题和昼夜模式切换，旨在提供简洁而高效的记录体验。

## 技术栈

- **开发框架**: .NET 8
- **UI 框架**: WPF (Windows Presentation Foundation)
- **编程语言**: C# 12
- **架构模式**: MVVM

## 开发环境

如果您想参与开发或本地构建：

- **IDE**: Visual Studio 2022 (推荐 17.8 或更高版本)
- **SDK**: .NET 8.0 SDK

## 发布包说明

在 GitHub Releases 中，我们提供两种类型的发布包，请根据您的环境选择：

### 1. 独立运行包 (win-x64-self-contained.zip)
- **描述**: 内置了完整的 .NET 8 运行时环境。
- **优点**: **开箱即用**，不需要在电脑上预先安装任何环境。
- **缺点**: 文件体积相对较大。
- **使用方法**: 下载并解压后，双击 `CyberNote.exe` 直接运行。

### 2. 依赖框架包 (win-x64-framework-dependent.zip)
- **描述**: 仅包含应用程序本身，不含运行时。
- **优点**: 文件体积非常小（通常几 MB）。
- **缺点**: 需要电脑上已安装 .NET 8 Desktop Runtime。
- **使用方法**:
    1. 检查或安装 [.NET Desktop Runtime 8.0](https://dotnet.microsoft.com/download/dotnet/8.0)。
    2. 下载并解压本程序的压缩包。
    3. 双击 `CyberNote.exe` 运行。

## 主要功能

- **多类型记录**: 支持普通随手记、任务清单（To-Do List）、富文本笔记。
- **活动统计**: 首页集成 GitHub 风格的活动热力图，直观展示每日记录频率与心情。
- **个性化主题**:
  - **颜色主题**: 内置绿、蓝、粉、橙、紫等多种主色调，一键切换。
  - **昼夜模式**: 完善的深色/浅色模式适配，夜间使用更护眼。
- **便捷交互**: 支持桌面悬浮球、系统托盘最小化、配置自动保存。
